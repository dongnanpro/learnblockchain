### ganache-cli 安装
testrpc 并入Truffle框架后，更名为：ganache-cli 
[ganache-cli](https://github.com/trufflesuite/ganache-cli)
```
npm install -g ganache-cli
```
运行
```
ganache-cli
```



### truffle安装
[官方安装指引](https://github.com/trufflesuite/truffle)

```
npm install -g truffle
```
